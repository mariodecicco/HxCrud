To use the API please use the following
GET http://[URL]/HxCrud/rest/users - returns a full list of users in XML format
GET http://[URL]/HxCrud/rest/users/[id] - return user information for the id provided
PUT http://[URL]/HxCrud/rest/users/ - pass in XML user and will update the user with data provided - returns full list of users
DELETE http://[URL]/HxCrud/rest/users/[id] - deletes the user provided from the list
POST http://[URL]/HxCrud/rest/users/ - pass in XML user to add
